Git y GitHub

Concepto de Git

Git es un sistema de control de versiones distribuidas. Esto significa que cada desarrollador tiene una copia completa del historial del proyecto en su equipos local y no depende de un servidor para trabajar.

Concepto de GitHub

GitHub es una plataforma en la web que permite almacenar y gestionar proyectos que utilizan Git como sistema de control de versiones. Fue lanzada en 2008 y adquirida por Microsoft en 2018. es la plataforma más popular en el mundo para alojar código, colaborar en proyectos de software y gestionarversiones demanera remotas.
